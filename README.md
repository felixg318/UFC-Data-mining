# UFC Fighter Outcome Prediction 

Predicting UFC fight outcomes using fighter statistics, physical attributes, and betting market data.

## Hypothesis

**Are betting market odds more accurate predictors of UFC fight outcomes than fighter statistics, physical attributes, and official rankings?**

## Dataset

- **Source**: `ufc-master.csv`
- **Link**: [Dataset source](https://www.kaggle.com/datasets/mdabbert/ultimate-ufc-dataset)

### Installation

1. Clone or download this repository

2. Install dependencies (recommended: in a virtual environment):
```bash
pip install -r requirements.txt
```

3. Launch Jupyter Notebook:
```bash
jupyter notebook DataMiningMilestone3.ipynb
```
